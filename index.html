Spring MVC do Jeito Certo

Aprenda a construir aplicações web robustas e flexíveis, aplicando boas práticas de programação e utilizando um framework completo de mercado que possui uma excelente implementação do padrão MVC e as funcionalidades que precisamos para atender as requisições HTTP, delegar responsabilidades de processamento de dados para outros componentes e preparar a resposta que precisa ser dada.

O que irei aprender?

Neste curso você aprenderá a criar a sua primeira aplicação web utilizando o Spring MVC do jeito certo. Para isso, demonstraremos como configurar o Spring MVC e criar um projeto simples explorando os recursos desse framework, programando, até mesmo, um cadastro de usuários em memória. Como você poderá notar, a relação desse framework com o padrão de projetos MVC faz dele uma das soluções Java mais adotadas para o desenvolvimento web, simplificando não apenas a organização do projeto.
